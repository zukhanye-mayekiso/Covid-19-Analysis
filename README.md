# Covid-19-Analysis
This project analyses COVID-19 case trends across South African provinces using Power BI, focusing on time-based trends, provincial comparisons, and data-driven insights.
Key Features:

Time-series analysis of total cases to identify peak periods and daily trends
Provincial comparison highlighting the most affected regions
Calculation of unresolved cases (Total Cases vs Unknown Cases)
Quarterly analysis to determine early outbreak patterns

Key Insights:

Gauteng recorded the highest number of total cases
Early spikes were observed in Gauteng during Q1 2020
Significant variation in impact across provinces

Tools Used:

Power BI
DAX
Data Modeling
